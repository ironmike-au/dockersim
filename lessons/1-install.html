Let's Install Docker

Before you can use docker, you need to install docker. No surpises here.

Docker runs on Linux, so you'll need a Linux server with root access. We use Ubuntu here, but most flavors will do just fine.

Don't have Linux handy? Fear not, my friend. There is a nifty tool called boot2docker that will install a tiny copy of Linux on a virtual machine on Windows or OS X, just for running Docker. It has everything you need.

But you can do all that stuff later. For now, let's get typing so you can see how to install Docker.

Install docker on Ubuntu with the following. Go ahead and type it in and see for yourself. 

sudo apt-get install docker.io

Woo! You did it. Docker is installed. But before we can use it, we need to download an image.

Docker images are pre-defined server environments complete with everything we might need.

You can build your own images too, but for now let's see how to get a ready-made one
